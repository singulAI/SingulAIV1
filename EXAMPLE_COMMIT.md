# Exemplo de Arquivo para Teste de Commit

Este arquivo foi criado para demonstrar como fazer commits seguindo os padrões do SingulAI Platform.

## Padrão de Commit Usado

```
feat(backend): add example file for commit demonstration
```

## Como Usar Este Exemplo

1. Faça mudanças em seus arquivos
2. Use: `git add .`
3. Commit com: `git commit -m "tipo(escopo): descrição"`

## Tipos de Commit

- `feat` - Nova funcionalidade
- `fix` - Correção de bug
- `docs` - Documentação
- `refactor` - Refatoração
- `test` - Testes
- `chore` - Manutenção

## Escopos

- backend
- contracts
- frontend
- docs
- infra
- project
